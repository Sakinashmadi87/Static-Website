Amazon Web Services (AWS)

AWS-Dienste

  • Amazon S3: Ein Objektspeicher-Service für die Speicherung und Sicherung von Daten.
  • Amazon EC2: Virtuelle Server für Rechenleistung und Skalierung.
    • Amazon EC2 Amazon EC2
    • Amazon EC2 Instance Types:
      • On-Demand:
        • Ideal für kurzfristige, unregelmäßige Workloads, die nicht unterbrochen werden können.
        • Keine Vorabkosten oder Mindestvertragslaufzeiten.
        • Pay-as-you-go-Preismodell.
      • Reserved Instances:
        • Standard Reserved Instance.
        • Convertible Reserved Instance.
      • EC2 Instance Saving Plans:
        • Commitment Duration and Flexibility.
        • Instance Family and Sizes.
        • Instance Flexibility.
        • Regional and Zonal Options.
        • Shared Usage.
        • 1-year or 3-year term.
      • Spot Instances:
        • Mögliche Unterbrechungen.
        • Bis zu 90 % Rabatt.
  • Amazon RDS: Ein verwalteter Datenbank-Service für relationale Datenbanken.
  • AWS Lambda: Ein Serverless Computing-Service für die Ausführung von Code ohne Serververwaltung.
